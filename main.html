    • URBAN CANVAS E-COMMERCE WEBSITE

    Transforming the online shopping experience, the Urban Canvas E-Commerce Website is a dynamic and feature-rich platform designed to bring the latest trends in home decor to your fingertips. With a seamless blend of technology and aesthetics, this project is a testament to my proficiency in web development and system architecture.

  • Responsive Design: Crafted a responsive user interface ensuring a seamless shopping experience across devices.
  • Add to Cart Functionality: Implemented an advanced shopping cart system for smooth and intuitive product selection.
  • Price Tracking: Incorporated a dynamic system to keep users informed about price changes.
  • Search Algorithm: Engineered a powerful search algorithm for swift and efficient product discovery.

    • TECHNOLOGIES USED

    Structural Foundation

    HTML lays the groundwork for the Urban Canvas E-Commerce Website, providing a clear and accessible structure for seamless user interaction.

    Aesthetic Enhancement

    CSS is employed to enhance the visual appeal of Urban Canvas, ensuring a cohesive and engaging design that reflects the brand's identity across various devices.

    Dynamic Interactivity

    JavaScript powers the dynamic features of Urban Canvas, enabling advanced shopping functionalities and real-time interactions, offering users an immersive and interactive online shopping experience.

    Backend Powerhouse

    Java, in synergy with Spring Boot, serves as the backend powerhouse of Urban Canvas, ensuring efficient integration, handling business logic, and contributing to a robust server-side infrastructure.

    Backend Efficiency

    MYSQL plays a vital role in managing the database for Urban Canvas, ensuring data integrity and scalability. It stores crucial information about products, user accounts.

    Data Management

    At the core of the backend, Spring Boot ensures efficient integration and handling of business logic, contributing to a robust server-side infrastructure.

    TOOLS USED

    In the development of the E-Commerce Website, Visual Studio Code serves as the primary code editor for handling HTML, CSS, JavaScript, and other front-end technologies. Its lightweight nature and swift performance make it an ideal choice for general-purpose coding tasks.

    POSTMAN API plays a crucial role in testing and debugging the backend APIs of the E-Commerce Website. This tool is employed to send requests to various API endpoints, allowing for the validation of their functionality and response.

    For the backend Java development aspect, particularly when working with the Spring Framework, SPRING TOOL SUITE takes center stage. STS is specifically designed to facilitate Spring-based development, providing specialized tools and features tailored for seamless integration with the framework.
